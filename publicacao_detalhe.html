Um Framework para Análise de Segurança em Aplicações Web

Silva, J., Costa, M., Oliveira, R. Anais do Simpósio Brasileiro de Segurança da Informação, 2025

Resumo

Este artigo apresenta um novo framework para a automação da análise de vulnerabilidades em aplicações web modernas. Com a crescente complexidade das tecnologias de front-end e back-end, métodos tradicionais de varredura se mostram insuficientes. Nosso framework, denominado WebGuard, utiliza uma combinação de análise estática de código (SAST) e análise dinâmica de aplicações (DAST) para identificar falhas de segurança comuns, como SQL Injection, Cross-Site Scripting (XSS) e Insecure Direct Object References (IDOR), com uma taxa de detecção significativamente superior às ferramentas existentes.

Introdução

O desenvolvimento de aplicações web tornou-se cada vez mais complexo com a adoção de frameworks JavaScript, APIs RESTful e arquiteturas de microsserviços. Essa complexidade introduz novas superfícies de ataque e vulnerabilidades que muitas vezes passam despercebidas pelos métodos tradicionais de teste de segurança.

Este trabalho propõe uma abordagem integrada que combina análise estática e dinâmica para fornecer uma avaliação mais abrangente da segurança de aplicações web modernas.

Metodologia

WebGuard foi desenvolvido utilizando uma arquitetura modular que permite a integração de múltiplas técnicas de análise:

  • Análise estática de código-fonte para detectar padrões vulneráveis
  • Varredura dinâmica de aplicações em tempo de execução
  • Teste de invasão automatizado para vulnerabilidades conhecidas
  • Análise de dependências de bibliotecas terceiras

Resultados

Em testes realizados com 50 aplicações web de código aberto, WebGuard demonstrou uma taxa de detecção de 94% para vulnerabilidades críticas, superando ferramentas comerciais como OWASP ZAP (78%) e Veracode (82%).

Além disso, o framework apresentou uma taxa de falsos positivos de apenas 6%, significativamente menor que as alternativas testadas.

Conclusão

WebGuard representa um avanço significativo na detecção automatizada de vulnerabilidades em aplicações web modernas. Sua abordagem híbrida que combina análise estática e dinâmica mostrou-se mais eficaz que soluções existentes no mercado.

Como trabalho futuro, planejamos expandir as capacidades do framework para incluir análise de aplicações single-page (SPA) e integração com pipelines de DevOps.
